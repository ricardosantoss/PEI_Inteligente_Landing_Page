# PEI Inteligente

Landing page estática para apresentar o projeto PEI Inteligente.

## Como publicar no Vercel

1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta para o repositório.
3. No Vercel, clique em **Add New Project**.
4. Importe o repositório do GitHub.
5. Deixe as configurações padrão e publique.

Como o projeto é estático, não precisa instalar dependências nem configurar build.

## Ajuste recomendado antes de publicar

No arquivo `index.html`, troque o e-mail do botão final:

```html
mailto:contato@peiinteligente.com.br
```

por um e-mail ou link de WhatsApp oficial do projeto.
